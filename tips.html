Douban Data Import Guide / 豆瓣数据导入指南

🇨🇳 中文指南

收藏夹淘金支持导入 JSON 格式的豆瓣已读、想读、看过、想看数据。您可以通过以下步骤快速获取并导入数据：

  1. 使用开发者推荐的开源工具 scrape-to-markdown.chrome。这是一个浏览器插件，专门用于抓取网页内容并导出为 Markdown 或 JSON。
  2. 安装插件后，访问您的豆瓣个人主页或相应的书单/影单页面，使用该插件一键抓取并导出数据为 JSON 格式。
  3. 在“收藏夹淘金”页面对应的 Tab 中点击相应的 “导入” 按钮，选择导出的 JSON 文件即可。

导入数据格式规范：

导出的 JSON 文件内容必须为数组格式，且包含以下属性：

[
  {
    "id": "37019075",  // 豆瓣条目 ID (选填)
    "title": "蒙特利尔，我的美人/Montréal, mabelle",  // 书名或电影名 (必填)
    "rating": "4星",  // 评分，支持 "4星"、"★★★★"、4 或 "力荐" 等 (选填)
    "readDate": "2026-04-29",  // 标记/阅读日期 (选填，格式如 YYYY-MM-DD)
    "comment": "陈冲真是她这个年龄层成就最大的华人女演员...",  // 你的短评或备注 (选填)
    "link": "https://movie.douban.com/subject/37019075/",  // 豆瓣详情链接 (选填)
    "type": "movie",  // 类别，如 movie 或 book (选填)
    "status": "collect"  // 状态，如 collect (看过/已读) 或 wish (想看/想读) (选填)
  }
]

🇬🇧 English Guide

Collection Miner supports importing Douban data (Read, Wishlist, Watched, To Watch) in JSON format. Follow the steps below to fetch and import your data:

  1. Install the developer-recommended open-source browser extension scrape-to-markdown.chrome. It is designed to scrape web content and export it as Markdown or JSON.
  2. Navigate to your Douban profile or book/movie status pages, and use the extension to fetch and export your records in JSON format.
  3. Click the "Import" button in the corresponding tab of the Collection Miner interface, and select the exported JSON file.

JSON Data Schema Specification:

The JSON file must contain an array of objects structured as follows:

[
  {
    "id": "37019075",  // Douban Item ID (Optional)
    "title": "Montréal, mabelle",  // Book or Movie Title (Required)
    "rating": "4星",  // Star Rating: supports "4星", "★★★★", 4, or "力荐" etc. (Optional)
    "readDate": "2026-04-29",  // Mark date (Optional, format: YYYY-MM-DD)
    "comment": "Joan Chen is indeed...",  // Personal review or note (Optional)
    "link": "https://movie.douban.com/subject/37019075/",  // Details Page URL (Optional)
    "type": "movie",  // Type: movie or book (Optional)
    "status": "collect"  // Status: collect (read/watched) or wish (to read/watch) (Optional)
  }
]
