# Collection Miner / 收藏夹淘金

## 中文说明

这是一个很简单的 Chrome 插件。
它会在你打开新标签页时，从你的收藏夹里随机挑 3 条出来给你看。
点开算一次访问，你每次看到它也会记录一次展示。

你可以点“再换三条”，继续随机看。
书签有新增或删除，它会自己更新。

## English

This is a very simple Chrome extension.
Every time you open a new tab, it randomly picks 3 bookmarks and shows them.
A click counts as a visit, and every display counts as a show.

You can hit "Refresh" to get 3 more.
If you add or remove bookmarks, it updates automatically.

## 安装 / Install

1. 打开 `chrome://extensions`
2. 开启开发者模式
3. 点击“加载已解压的扩展程序”
4. 选择这个文件夹
