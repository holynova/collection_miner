# Collection Miner / 收藏夹淘金

## 中文说明

这是一个很简单的 Chrome 插件。
它会在你打开新标签页时，从你的收藏夹里随机挑 3 条出来给你看。
点开算一次访问，你每次看到它也会记录一次展示。

你可以点“再翻三张”，继续随机看。
书签有新增或删除，它会自己更新。

现在的卡片支持：
- 整张卡片可点击打开链接
- 点赞（红心），点过会变成实心
- 右下角垃圾桶可以直接删除书签（可撤销）
- 显示添加时间（年月日）和距今时长
- 赞/踩会影响下一次的随机权重

## English

This is a very simple Chrome extension.
Every time you open a new tab, it randomly picks 3 bookmarks and shows them.
A click counts as a visit, and every display counts as a show.

You can hit "Refresh" to get 3 more.
If you add or remove bookmarks, it updates automatically.

Card features:
- The whole card is clickable to open the link
- Like button (heart), toggles filled state
- Trash icon deletes the bookmark (undo supported)
- Shows added date and time since added
- Likes/dislikes affect future random picks

## 安装 / Install

1. 打开 `chrome://extensions`
2. 开启开发者模式
3. 点击“加载已解压的扩展程序”
4. 选择这个文件夹
