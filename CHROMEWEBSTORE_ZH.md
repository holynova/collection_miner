# Chrome 应用商店上架指南 — 收藏夹淘金 (Collection Miner)

> 最后更新日期：2026-06-25

## 商店上架信息 (Store Listing)

**扩展程序名称 (Extension Name)**
收藏夹淘金

**简短说明 (Short Description)** (最多 132 个字符)
每次打开新标签页，都能重新发现一条被遗忘的书签。配合智能权重推荐系统。

**详细描述 (Detailed Description)**
“收藏夹淘金 (Collection Miner)”将您的新标签页变成一场每日探宝之旅，带您重新挖掘自己尘封已久的书签库。

每次您打开新标签页，插件都会从您的浏览器书签库中随机挑选 3 条。随着您与卡片的持续交互（点赞、点击或点踩），系统会动态学习您的偏好并调整权重，让您真正喜欢的书签出现频率越来越高，而您不感兴趣的则会逐渐淡出。

核心功能特性：

书签淘金 (Bookmark Mining)：每次打开新标签页时，从您的完整书签库中随机展示 3 条书签。

动态权重系统 (Dynamic Weight System)：点赞卡片可提高其未来出现的概率，点踩则会降低其出现概率。插件会记录展示次数、点击次数和赞踩评分，随时间推移越来越懂您的偏好。

视觉稀有度分级 (Rarity Tiers)：根据您的评分，卡片将获得精致的稀有度边框光晕——`传说 Legendary`（金色光晕）、`稀有 Rare`（紫色/蓝色光晕）、`普通 Common`（白色边框）、`诅咒 Cursed`（暗淡效果）。

历法时间分级 (Age Tiers)：卡片还会根据书签被收藏的长短呈现不同的金属光泽——`金`（收藏 5 年以上）、`紫`（3 年以上）、`蓝`（2 年以上）、`绿`（1 年以上）、`白`（1 年以内）。

极具质感的卡片设计 (Premium Card Design)：卡片支持鼠标跟随的 3D 悬浮倾斜效果，配备跟随光晕效果，精致的 3D 翻转刷新动画，以及三种精美主题卡背样式（皇家经典 Royal Classic、赛博霓虹 Cyber Neon、星空星座 Celestial Constellation）。

双语界面 (Bilingual UI)：支持中英文界面一键切换。

快捷刷新 (Keyboard Shortcut)：按下 `Enter` 键或空格键即可快速“再翻三张”。

数据本地备份 (Data Backup)：支持随时将卡片权重、历史记录一键导出为本地 JSON 文件备份，或导入备份进行恢复。

隐私安全 (Privacy)：所有数据均保存在您的本地设备上。不收集、不传输、不共享任何用户隐私数据。有关详情请参阅我们的隐私政策页面。

使用指南：安装插件并打开一个新标签页，您的书签会立刻呈现。

如遇问题或有任何反馈，欢迎通过插件内关联的 GitHub 仓库提交反馈。

## 权限使用声明 (Permissions Justification)

| 所需权限 | 权限类型 | 权限合理性说明 |
|------------|------|---------------|
| bookmarks | permissions | 插件需要读取用户的浏览器书签，以便在每次打开新标签页时随机挑选并展示 3 条。如果没有此权限，插件的核心功能将无法运行。书签数据仅在本地读取，绝不会被修改或传输。 |
| storage | permissions | 插件使用 Chrome <code>storage</code> API 在本地保存书签的展现权重（赞/踩次数、历史展现记录）以及用户的个性化配置（选定布局、卡背主题、语言）。此数据完全本地保存，绝不上传至任何服务器。 |
| tabs | permissions | 插件需要读取当前打开标签页的 URL，以便在检测到重复打开“收藏夹淘金”标签页时，提供一键“关闭其他”按钮。插件不读取、不存储任何网页内容或浏览历史。 |
